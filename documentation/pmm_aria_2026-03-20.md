# Pmm Aria / MyISAM

## Route / Code

- Route: `/Pmm/aria/{id_mysql_server}`
- Controller action: `App/Controller/Pmm.php::aria()`
- View: `App/view/Pmm/aria.view.php`
- Shared view shell: `App/view/Pmm/dashboard.view.php`
- Frontend: `App/Webroot/js/Pmm/dashboard.js`

## PMM Source

- `dashboards/MySQL/MySQL_MyISAM_Aria_Details.json`

## Implemented Sections

### Aria pagecache

PMM equivalent panels:

- Aria Metrics
- Aria Pagecache Reads/Writes
- Aria Pagecache Blocks

PmaControl charts:

- line: Aria pagecache blocks
- line: Aria pagecache read pressure
- line: Aria hit ratio

Metrics:

- `aria_pagecache_buffer_size`
- `aria_log_file_size`
- `aria_block_size`
- `aria_pagecache_blocks_used`
- `aria_pagecache_blocks_unused`
- `aria_pagecache_blocks_not_flushed`
- `aria_pagecache_reads`
- `aria_pagecache_read_requests`

### MyISAM key cache

PMM equivalent panels:

- MyISAM Metrics
- Key Buffer Performance
- Key Cache

PmaControl charts:

- line: MyISAM key blocks
- line: MyISAM key cache activity
- line: MyISAM key cache hit ratio

Metrics:

- `key_buffer_size`
- `key_blocks_used`
- `key_blocks_unused`
- `key_read_requests`
- `key_reads`
- `key_write_requests`
- `key_writes`
- `myisam_sort_buffer_size`
- `myisam_recover_options`
- `delay_key_write`

## PMM Metric Origins

Main PMM origins:

- `mysqld_exporter`
- MySQL/MariaDB global status and variables

## Equivalent In PmaControl

- Aria and MyISAM values are already collected in PmaControl and exposed through `Extraction2`
- rates and hit ratios are rebuilt server-side from cumulative counters
